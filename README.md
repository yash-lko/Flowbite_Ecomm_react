A modern e-commerce website built using **React.js** with real-world shopping features.

## 🚀 Features
- Product listing fetched from API
- Add to cart functionality
- Quantity increase/decrease
- Automatic total price calculation
- Responsive UI

## 🛠 Tech Stack
- React.js
- JavaScript (ES6)
- Tailwind CSS / CSS
- REST API
